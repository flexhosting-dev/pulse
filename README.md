# Pulse

**Stay in rhythm with your goals**

Pulse is a modern Performance Management System built with Symfony, designed for continuous feedback and growth.

## About

Pulse replaces traditional annual-review-only systems with a continuous performance management approach that includes:

- Goal setting with SMART objectives
- Regular check-ins and monitoring
- Mid-year and annual appraisals
- 360-degree feedback surveys
- Performance improvement plans
- Counselling and coaching documentation

## Tech Stack

- **Backend:** Symfony 7.x (PHP 8.3+)
- **Database:** PostgreSQL
- **Frontend:** Symfony UX (Turbo + Stimulus), Bootstrap 5
- **Workflow:** Symfony Workflow Component

## Features

### Core Modules
- Dashboard with analytics
- Annual KPIs/KRAs management
- Mid-Year Performance Appraisals
- Annual Performance Appraisals
- 360 Performance Survey
- Performance Improvement Plan (PIP)
- Continuous Monitoring (Check-ins & Counselling)

### Key Capabilities
- SMART goal validation
- Weighted objectives with auto-calculation
- Multi-level approval workflows
- Audit trail and documentation
- PDF/Excel export
- Email notifications
